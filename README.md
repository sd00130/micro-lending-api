# Micro-lending rest api

# How to run:
```
git clone https://github.com/sd00130/micro-lending-api.git
```
```
cd micro-lending-api
```
```
mvn clean install
```
For integration tests:
```
mvn clean install -Pitest
```
```
mvn spring-boot:run
```
Swagger at:
```
http://localhost:8080/swagger-ui.html
```
